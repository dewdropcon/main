# DEW Drop Construction Company Website

## Project Overview

This repository hosts the official website for **DEW Drop Construction Company**, based in Marondera, Zimbabwe. We specialize in high-quality painting, plumbing, and tiling services, and are proficient in a wide range of general construction works. Our team of intuitive construction workers and specialists is dedicated to transforming your vision into reality with durable and aesthetically pleasing results.

The website serves as an online presence to showcase our services, provide company information, and offer a direct contact channel for potential clients.

## Features

*   **Home Section:** Introduction to DEW Drop Construction Company.
*   **About Us:** Detailed information about the company's mission, values, and expertise.
*   **Services:** Comprehensive overview of our core services, including:
    *   Painting
    *   Plumbing
    *   Tiling
    *   General Construction
*   **Contact Us:** A contact form for inquiries and direct contact information (phone, email, location).
*   **Responsive Design:** Optimized for viewing on various devices (desktops, tablets, and mobile phones).
*   **Smooth Navigation:** Interactive elements for an enhanced user experience.

## Technologies Used

*   **HTML5:** For structuring the content of the website.
*   **CSS3:** For styling and layout, ensuring a modern and appealing design.
*   **JavaScript:** For interactive elements and dynamic content (e.g., smooth scrolling, dynamic year in footer).

## Setup and Usage

To view the website locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/dewdrop.github.io.git
    ```
    (Note: Replace `your-username` with the actual GitHub username if this project is hosted there.)
2.  **Navigate to the project directory:**
    ```bash
    cd dewdrop.github.io
    ```
3.  **Open `index.html`:** Simply open the `index.html` file in your preferred web browser.

## Deployment

This website is designed for static hosting. It can be easily deployed to platforms such as:

*   **GitHub Pages:** The `_config.yml` file suggests compatibility with Jekyll, commonly used for GitHub Pages.
*   **Netlify**
*   **Vercel**
*   Any other static site hosting provider.
